Question Form User Guide

Welcome! This web application is the first step in enabling you to build your very own mobile application.

In order to do this, you can use this application to create user-specified questions at desired locations which will appear on the mobile app in their respective locations.



Key information:

- Please ensure the httpServer.js is running.

- Please ensure all fields within the question form are populated.

- Please find more basemap layer selections available by selecting the layers button at the top right of the map

- When selecting the 'View Questions' button, ensure the blue location marker has been changed from previously submitted question location otherwise you will not be able to select the marker to drag it further. Additionally, selecting the 'Home' button will reset the map and therefore remove the 'View Questions' layer.





Generating a new question:

1: Firstly, the location of the question can be selected by clicking and dragging the blue marker in the map to a desired location.

2: Once location is selected, release the marker and you will see the coordinates in the latitude and lontidue fields to the right of the map have been automatically filled with the coordinates corresponding to the marker position.

3: You can now populate the remaining fields within the form to the right of the map (i.e. Questions, and Answers etc.)

4: That question may now be submitted using the "Submit" button below.

- Please ensure all fields in the form and populated, otherwise the question will not be allowed to submit.

5: Once the question has been submitted, you can now select the "View Questions" button to the left of the map, creating a layer of all questions successfully submitted to the database.

6: You can now use the "Clear Question" button on the left of the map to reset the question form and restart the process to submitting another question.
